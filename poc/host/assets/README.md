# Embedded font

`DejaVuSansMono.ttf` — DejaVu Sans Mono, from the DejaVu Fonts project.

DejaVu fonts are released under a free license (a Bitstream Vera / public-domain
derived license) that permits embedding and redistribution. It is bundled here so
the host renders text identically everywhere, independent of system fonts — which
is itself a user-agency property the paper calls out (text rendering belongs to
the engine, not userland).
